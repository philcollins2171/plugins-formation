# Clients/

Un sous-dossier par client, structure identique pour chacun :

```
[NomClient]/
├── knowledge.md   ← contexte stable (interlocuteurs, pricing, conventions)
├── history.md     ← événements datés (réunions, décisions, livraisons)
└── livrables/     ← fichiers produits pour ce client
```

## Créer un nouveau client

Copier le dossier `_template/` et renommer.
